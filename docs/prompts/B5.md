## 1. `src/components/BrandLogo.tsx` — Full Rewrite

### New Logo Concept

- **Icon:** Small amber geometric mark to the LEFT of the wordmark. Abstract shape suggesting automation/flow — NOT a figure-8, NOT an infinity symbol. Use an abstract angular/geometric shape: three interlocking chevrons or a stylized circuit node. Simple enough to work at 16x16 favicon size.
- **Wordmark:** `Autom8` in Manrope 700, warm off-white (`#EDEBE8`). The `8` character should be rendered with an amber gradient (`#D4A030` → `#E8C068`).
- **Descriptor:** `everything` appears below the wordmark in Inter 600 uppercase tracking-wide, `#9B978F` color. Only shown when `showDescriptor` is true.
- **The icon uses an inline SVG** with amber fill (`#D4A030`).

### Implementation Spec

```tsx
// BrandMark — geometric icon
// SVG viewBox="0 0 32 32"
// Design: Three stacked chevrons pointing right, slightly offset vertically
// Fill: amber gradient from #D4A030 to #E8C068
// Container: rounded-lg bg with --accent-subtle background, amber glow shadow on hover

// BrandWordmark
// "Autom" in Manrope 700, color #EDEBE8
// "8" in Manrope 700 with amber gradient (bg-gradient, bg-clip-text, text-transparent)
// "everything" descriptor: Inter 600, uppercase, tracking-[0.22em], color #9B978F
```

### Size Classes (keep same structure)
| Size | Mark | Text | Descriptor | Gap |
|------|------|------|-----------|-----|
| xs | h-6 w-6 | text-base | text-[9px] | gap-2 |
| sm | h-8 w-8 | text-lg | text-[10px] | gap-2.5 |
| md | h-10 w-10 | text-xl | text-xs | gap-3 |
| lg | h-14 w-14 | text-3xl | text-sm | gap-3 |

### SVG Icon Design

Create an inline SVG with viewBox="0 0 32 32". Design: abstract automation flow icon — three parallel diagonal lines or chevrons arranged to suggest forward motion/flow. Keep it geometric and clean. Colors:

```xml
<defs>
  <linearGradient id="autom8-mark-gradient" x1="0%" y1="0%" x2="100%" y2="100%">
    <stop offset="0%" stop-color="#D4A030" />
    <stop offset="100%" stop-color="#E8C068" />
  </linearGradient>
</defs>
```

The mark container should have:
- `bg-[rgba(212,160,48,0.10)]` (accent-subtle)
- `rounded-xl`
- `border border-white/10`
- `shadow-[0_0_20px_rgba(212,160,48,0.12)]`

### Font References

ALL `var(--font-playfair)` references → `var(--font-manrope)`. The wordmark uses `style={{ fontFamily: "var(--font-manrope), sans-serif" }}`.

### Keep

- Same component API (props: href, className, size, showDescriptor, showMark, as, screenReaderText)
- Same Link/span rendering logic
- Same accessibility (sr-only text, aria-label)
