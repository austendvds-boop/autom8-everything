Every section's heading block (h2 + subheading) must use scroll-triggered reveal:

```tsx

  initial={prefersReducedMotion ? false : "hidden"}
  whileInView="visible"
  viewport={viewportOnce}

  <h2>...</h2>
  <p>...</p>
</motion.div>
```

## Pattern 2: Card Stagger

For any grid of cards (ServicesBento products, WhoItsFor industries, Testimonials quotes, HowItWorks steps):

```tsx
// Container
<motion.div
  className="grid ..."
  variants={staggerContainer}
  initial={prefersReducedMotion ? false : "hidden"}
  whileInView="visible"
  viewport={viewportOnce}
>
  {items.map((item, i) => (
    <motion.div key={...} variants={staggerItem}>
      {/* card content */}
    </motion.div>
  ))}
</motion.div>
```

The `staggerContainer` uses `staggerChildren: 0.1, delayChildren: 0.05`. The `staggerItem` fades up from y:30 with scale:0.97. This creates the signature Framer "cards revealing one by one" effect.

## Pattern 3: Card Hover (Lift + Glow)

Every card should have hover animation. Use `cardHover` from motion.ts on the motion element:

```tsx
<motion.article
  {...cardHover}
  className="... hover:shadow-[0_0_40px_rgba(212,160,48,0.20)]"
>
```

The CSS `hover:shadow` handles the amber glow. The framer-motion `cardHover` handles the y:-4 lift. Together they create the "alive" hover state.

Additionally, card borders should transition on hover:
```
border border-white/[0.06] hover:border-[rgba(212,160,48,0.30)]
transition-all duration-300
```

## Reduced Motion (MANDATORY)

Every component must call `useReducedMotion()` from framer-motion. When reduced motion is preferred:
- All whileInView animations → simple `opacity: 0 → 1` (use `revealReduced`)
- No y/x transforms, no scale transforms
- No hover transforms (cardHover, buttonHover should be skipped)
- No gradient drift (static gradient)
- No stagger delays
- FAQ accordion can still animate (it's functional, not decorative)

Pattern:
```tsx
const prefersReducedMotion = useReducedMotion();

// For reveals:
{...(prefersReducedMotion ? revealReduced : reveal)}

// For hover:
{...(prefersReducedMotion ? {} : cardHover)}
{...(prefersReducedMotion ? {} : buttonHover)}

// For stagger container:
initial={prefersReducedMotion ? false : "hidden"}
```
