## Universal Color Replacement Rules (apply to ALL four files)

### Hex Colors
| Old | New | Notes |
|-----|-----|-------|
| `#8B5CF6` | `#D4A030` | Primary accent |
| `#A78BFA` | `#E8C068` | Secondary accent |

| `#C4B5FD` | `#E8C068` | Light accent |
| `#A1A1AA` | `#9B978F` | Text secondary |
| `#71717A` | `#5E5B56` | Text muted |
| `#52525B` | `#5E5B56` | Text muted alt |
| `#D4D4D8` | `#EDEBE8` | Light text |
| `#111118` | `#161920` | Surface bg |

| `#0F0F18` | `#0E1015` | Primary bg alt |

| `#12121A` | `#161920` | Surface bg alt |
| `#F59E0B` (stars only) | `#F59E0B` | Keep — star color stays |

### RGBA Values
| Old Pattern | New Pattern |
|-------------|------------|
| `rgba(139, 92, 246, X)` | `rgba(212, 160, 48, X)` — same opacity |
| `rgba(6, 182, 212, X)` | `rgba(212, 160, 48, X)` — at half opacity (reduce visual weight) |
| `rgba(167, 139, 250, X)` | `rgba(232, 192, 104, X)` — same opacity |

### Font Family
All instances of `var(--font-playfair)` → `var(--font-manrope), sans-serif`

### Gradient Text
If any `<span className="gradient-text">` exists — keep it. The `.gradient-text` class now uses amber gradient (set in B4 globals.css).
